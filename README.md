File Packer Unpacker (Java)

A simple Java application that allows you to **pack multiple files into one** and **unpack them back**.  
Useful for combining files into a single archive format.

---

 Features
- Pack all files from a directory into one file  
- Unpack all files from the packed file  
- Displays file names and sizes during operations  
- Simple console-based menu  
- Supports any file type
